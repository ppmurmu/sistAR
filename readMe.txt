sistAR getting started:


sistAR consists of two parts:
1. sistAR mobile for workers on the shop floor
2. sistAR web for managers


The installation process for sistAR:
1. Download the app-release.apk which is the main app. 
2. Download the sistaCam.apk which is the support unity app for the main sistAR app.
3. Install both of them and then open the sistAR app.
4. Voila! You’re ready to go. Use any of the below username and passwords to access.
